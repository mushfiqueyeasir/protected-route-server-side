Welcome to Authentication Server

Authentication Server API is a JSON-based auth enabled API. All requests are made to endpoints beginning: https://protected-route-server-side.vercel.app/api/v1 All requests must be secure, i.e. https, not http.

    API URL
  • Click here to make a Post request to add User information
